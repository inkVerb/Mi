### Dark theming on Chromium browsers

- Chrome
- Chromium
- Vivaldi
- Edge

#### Flag setting

| **Paste into browser** ://

```console
chrome://flags/#enable-force-dark
```

*Set **Auto Dark Mode for Web Contents** to "Enabled with selective inversion of non-image elements"*
